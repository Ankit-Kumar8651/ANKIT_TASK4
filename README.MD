# Card Layout Project

## Description
This project demonstrates the use of viewport units and the CSS box model to create a responsive card layout.

## Features
- Responsive design using vw and vh units
- Proper use of box-sizing
- Styled button with gradient and hover effects
- Clean and simple UI

## How to Run
1. Download the project files
2. Open index.html in any browser

## Technologies Used
- HTML5
- CSS3
